# Youtube Manager

A simple, robust command-line application to manage video entries (CRUD) stored in JSON.

**Features**
- List, Add, Update, Delete videos
- Stable integer `id` for each video
- Safe JSON persistence (atomic writes, corrupted-file backup)
- Input validation and helpful prompts
- Search & sort helpers
- Logging for diagnostics
